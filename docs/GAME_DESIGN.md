# Gimcheon (가제) — 게임 디자인 문서

> 1대1 오토배틀러 (롤토체스 느낌). 준비 → 전투 → 정산을 반복하는 라운드제 전략 게임.
> 이 문서는 **시스템/규칙**을 정의합니다. 종족·유닛 콘텐츠는 [RACES.md](RACES.md), 코드 구조·데이터 스키마는 [ARCHITECTURE.md](ARCHITECTURE.md) 참고.

표기: `🔶 확인 필요` = 원 스펙에 없어서 제가 제안한 값/규칙. 보시고 승인/수정해 주세요.
모든 수치는 **초기 밸런스 기준값**이며 `data/config.ts`에서 한 곳에서 조정합니다.

## 1. 개요 / 비전

- **장르**: 1대1 라운드제 오토배틀러
- **한 판 흐름**: 준비기간에 병력 구매·건물 업그레이드·연구 → 자동 전투 → 코인 정산 → 다음 준비기간 … 반복
- **승리 조건**: 상대의 목숨 6개를 모두 소진시키면 승리
- **종족**: 자연 / 인간 / 마계 (각자 다른 핵심 메커니즘과 발전 방향)
- **1차 범위**: 솔로 모드(= AI 상대) 만. 멀티 1대1은 이후. 단, 코드는 처음부터 멀티 확장을 전제로 설계(전투 시뮬레이션을 순수·결정론적으로 분리).

## 2. 핵심 루프 & 타이밍

| 단계 | 시간 | 내용 |
|---|---|---|
| 준비기간 (Prep) | 60초 | 병력 구매, 건물 업그레이드, 연구, 그리드 배치, 발전 방향 선택(6턴~) |
| 전투 (Battle) | 30초 | 배치된 유닛이 자동 전투 |
| 연장전 (Overtime) | 10초 | 미결착 시. **모든 유닛 공격력 ×2, 공격속도 ×2** (강제 결착) |
| 정산 (Settlement) | 즉시 | 코인·보석 수급, 목숨 차감, 다음 준비기간 시작 |

- 1 사이클 = **1턴**.
- 준비기간 타이머가 끝나면 자동으로 전투 시작(플레이어가 일찍 "준비완료"로 단축 가능). `🔶 확인 필요`: 조기 준비완료 시 남은 시간 보너스 줄지 여부 — 기본은 보너스 없음.

## 3. 목숨 & 승패

- 각 플레이어 목숨 **6개**. 0이 되면 패배.
- **전투 패배 시 목숨 -1.** (단순·명확) `🔶 확인 필요`
  - 대안 A(차등): `잃는 목숨 = 1 + (살아남은 적 유닛 수에 비례한 추가 피해)`. 빠른 게임·스노우볼 원하면 채택.
  - 대안 B(고정): 위처럼 항상 1.
  - → **기본은 대안 B(항상 1)** 로 구현하고, config 플래그로 A 전환 가능하게 둠.
- 무승부(연장전 종료 시 정확히 동률): 양쪽 목숨 변화 없음.
- 연장전 종료 판정: **남은 아군 유닛들의 잔여 HP 총합이 큰 쪽 승리.** 동률이면 무승부. `🔶 확인 필요`

## 4. 그리드 & 배치

- 전장은 하나의 그리드. **위 절반 = 상대 진영, 아래 절반 = 내 진영.** (롤토체스식)
- 배치는 **내 진영 칸에만**. 기본 **한 칸당 한 유닛**.
- **자연 종족 겹침 배치(스택)**: 자연은 개체 능력치가 낮은 대신 물량 종족이라, **발전 방향을 선택한 뒤(7턴~)부터 한 칸에 최대 3유닛까지 겹쳐 배치** 가능. 다른 종족 및 발전 전 자연은 1유닛/칸. (전투 중 겹침 처리는 §5)
- 그리드 크기(초기안): **가로 7 × 세로 8** (내 진영 4행 = 28칸). 출전 상한 20을 무난히 수용. `🔶 확인 필요`
- 전투에 내보낼 수 있는 유닛 수는 **전투 인원 증가 시설** 레벨로 제한, **최대 20** (9. 공통 건물 참고).
- 전투가 시작되면 유닛은 칸을 벗어나 연속 좌표로 이동하며 난전. 단 **충돌 반경**으로 서로 밀어내 겹치지 않음(5. 참고).

## 5. 전투 시뮬레이션 규칙

전투는 **고정 timestep 틱 기반**으로 결정론적으로 진행합니다. (멀티/리플레이 대비)

- 틱: **30 tick/초** (`dt = 1/30s`). 전투 30초 = 900틱, 연장 10초 = 300틱.
- 모든 무작위 요소는 **시드 RNG** 하나로만. 유닛 처리 순서는 항상 `unitId` 오름차순(결정론).

**유닛 1틱 처리 순서**
1. 타깃이 죽었거나 무효면 해제.
2. 타깃 없으면 **가장 가까운 공격 가능한 적**을 획득(거리 동률 시 더 작은 id 우선).
3. 타깃이 **사거리 안**이면: 공격 쿨다운(`1/공격속도`초, 연장전엔 절반)이 찼을 때 공격.
4. 사거리 밖이면: 타깃 방향으로 `이동속도`만큼 이동(+ 분리 스티어링).
5. 전체 이동 후 **원형 충돌 해소**(서로 반경만큼 밀어내기, 1~2회 반복)로 겹침 제거.

**자연 겹침(스택) 처리** `자연 전용`
- 같은 칸에 겹쳐 배치한 자연 아군은 **동일 `stackId`** 를 공유하고, **그 그룹 내에서는 상호 충돌(밀어내기)을 무시**해 겹친 상태로 함께 이동.
- 칸 용량이 3이므로 한 그룹 = 최대 3유닛 → **겹침은 최대 3까지만**. 서로 다른 스택끼리, 그리고 적·타 종족과는 정상 충돌.
- 효과: 좁은 공간에 화력 집중·전열 강화. 트레이드오프: **광역기에 취약**(뭉쳐 있어 한 번에 피해).

**레이어 & 타게팅 규칙**

| 공격자 ↓ \ 대상 → | 지상 | 공중 | 땅속(underground) |
|---|---|---|---|
| 지상 근접(사거리1) | ✅ | ❌ | ❌ |
| 원거리/대공 플래그(`canHitAir`) | ✅ | ✅ | ❌ |
| `canHitUnderground` 보유 | ✅ | (플래그 따름) | ✅ |
| 공중 유닛 | ✅ | ✅ | ❌(기본) |

- **공중**: `canHitAir`를 가진 유닛에게만 피격. 지상 근접은 공중을 못 때림.
- **땅속**(자연 "땅 밑 친구들" 발전): 기본적으로 `canHitUnderground` 유닛에게만 피격하며, 이동 시 충돌을 무시(다른 유닛 밑으로 통과). `🔶 확인 필요`: 고급안은 "공격할 때 잠깐 지상으로 부상하여 그 순간 피격 가능"(매복 뉘앙스). v1은 단순안, 부상 메커니즘은 옵션.

**투사체**: 원거리 유닛은 v1에서 **즉시 명중 + 투사체 비주얼**(시뮬엔 즉시 적용). 추후 실투사체(이동·회피) 옵션화.

## 6. 능력치 & 데미지 공식

각 유닛 능력치: **HP / 공격력(ATK) / 공격속도(초당 공격수) / 이동속도(초당 칸) / 사거리(칸) / 방어력(DEF)**.

- **데미지 = `ATK_eff × 100 / (100 + DEF) `** (방어력 = 감산%형 아머), 최소 1.
  - 예: DEF 50 → 받는 피해 `100/150 ≈ 67%`. DEF 100 → 50%.
  - `🔶 확인 필요`: 감산%형(추천, 밸런스 안정) vs 고정 감산형(`max(1, ATK-DEF)`). 기본은 감산%형.
- **연장전**: `ATK_eff = ATK×2`, 공격속도 `×2` → 실효 DPS 약 ×4. 강제 결착용.
- 초기 능력치 스케일(가이드라인):

| 능력치 | 범위(대략) |
|---|---|
| HP | 250 ~ 2000 |
| ATK | 15 ~ 120 |
| 공격속도 | 0.5 ~ 1.5 /초 |
| 이동속도 | 1.5 ~ 4.2 칸/초 |
| 사거리 | 1(근접) ~ 5 |
| 방어력 | 0 ~ 50 |
| 충돌 반경 | 0.3 ~ 0.6 칸 |

## 7. 경제 — 코인 정산

코인은 **병력 구매 / 건물 업그레이드 / 발전 해금 / (인간)시대 진화 / (자연)부화장·(마계)프리즘 해금** 등에 쓰는 주 재화.

매 턴 정산 코인(초기안, 모두 `🔶 확인 필요`이며 config로 조정):
- **기본 수입**: +5
- **이자**: 보유 코인 10당 +1, 최대 +5 (경제 운영 유도)
- **승리 보너스**: +1 (패배 시에도 +1 자비 수입)
- **연승/연패 보너스**: 2/4/6 연속에서 +1/+2/+3
- 시작 코인: 3

→ 초반엔 턴당 ~1–2유닛 구매 페이스. 저축(이자) vs 즉시 전력 강화의 트레이드오프.

`🔶 확인 필요`: 병력 구매 방식
- **(기본) 직접 구매**: 해금된 유닛을 정가에 원하는 만큼 구매(코인·인원 상한 내). 결정론적·전략적.
- (대안) 롤토체스식 랜덤 상점 + 새로고침. RNG 추가. — 유닛이 고정가/소수라 기본은 직접 구매 채택.

## 8. 보석 & 연구 (연구소)

- **보석(Gem)**: 연구 전용 재화. 코인과 별개.
- **수급(초기안)** `🔶 확인 필요`: 연구소가 매 턴 보석 생산. `보석/턴 = 2 + 연구소 레벨` (+승리 시 +1). → 연구소 업그레이드(코인)가 보석 경제 투자가 됨.
- **연구소 기능**: 유닛별 연구 트리에 보석을 투자.
  - **기본 능력치 향상**: 노드별로 HP/ATK 등 +가산. 레벨이 오를수록 비용 증가.
  - **추가 능력 해금**: 특정 능력치를 임계치 이상으로 올리면 그 유닛의 특수 능력 해금(예: 공격력 3렙 이상 → 특수 능력 활성).
- 연구는 **판 사이에 영속? 아니면 판 한정?** `🔶 확인 필요`: 1차는 **한 판(매치) 내 한정**(매치 시작 시 초기화). 메타 진행/영속 업그레이드는 이후.

연구 노드 데이터 구조는 [ARCHITECTURE.md](ARCHITECTURE.md) 참고.

## 9. 공통 건물 (모든 종족)

| 건물 | 역할 | 업그레이드 효과(초기안) |
|---|---|---|
| **병력 생산 시설** | 기본 유닛 생산/구매 허브 | 레벨이 구매 가능한 유닛 **티어**를 해금(고가 유닛은 상위 레벨 필요). `🔶 확인 필요` |

| **전투 인원 증가 시설** | 출전 가능 유닛 수 상한 | Lv1 = 8명 시작, 레벨당 +2, **최대 20명** (시작값/증가량은 튜닝 가능) |

업그레이드 비용/곡선은 `data/buildings.ts`에서 정의.

## 10. 발전 방향 (6턴 이후)

- **7턴 진입 시(=6턴 종료 후)** 2개 발전 방향 중 **1개를 영구 선택**.
- 각 종족 2지선다 (자세한 유닛은 RACES.md):
  - 자연: **고대**(공룡형 4유닛) / **땅 밑 친구들**(땅속 4유닛)
  - 인간: **첨단**(레이저·메크 4유닛, 일반 유닛 합체로 소환, 일정 시대 이상 필요) / **마법**(마법사·골렘 4유닛, 중세 이전이어야 선택 가능)
  - 마계: **1군단장**(근접 군단장) / **2군단장**(원거리 마법 군단장)
- 인간의 시대 게이팅은 **상호 배타적 빌드 압박**을 만든다: 첨단=시대를 빨리 올려야, 마법=중세 이전에 머물러야 → 초반 시대 운영이 발전 방향을 결정. (의도된 핵심 설계)

## 11. 솔로 모드 (AI 상대)

- AI는 매 준비기간에: 예산 내 유닛 구매(탐욕적, 역할 균형) → 그리드 배치 → 정해진 스케줄로 건물 업그레이드/연구 → 6턴에 발전 방향 선택.
- 난이도는 **예산 배율 + 의사결정 품질**로 조절(쉬움/보통/어려움).
- `AiController` 인터페이스로 분리해 추후 교체/고도화 가능. (ARCHITECTURE.md)

## 12. 확인이 필요한 열린 결정 (요약)

아래만 확정해 주시면 M1 구현으로 들어갑니다. (각 항목 옆 제 기본 제안)

> ✅ 확정 반영됨: 출전 상한 **20**, 자연 종족 **겹침 배치**(발전 후 한 칸 최대 3유닛). — §4, §5, §9, RACES.md 반영 완료.

1. **목숨 차감**: 패배 시 항상 -1 *(기본)* / 살아남은 적 수 비례 차등?
2. **유닛 구매**: 직접 구매(정가) *(기본)* / 롤토체스식 랜덤 상점?
3. **보석 수급**: 연구소가 매 턴 생산(`2+레벨`) *(기본)* / 다른 방식?
4. **연구 영속성**: 매치 한정 *(기본)* / 판 사이 영속?
5. **그리드 크기**: 7×8 *(기본)* / 다른 크기?
6. **출전 인원**: **최대 20 확정**(사용자 지정). 시작 8 → 레벨당 +2 → 20 제안 (시작값/증가량만 확인).
7. **데미지 공식**: 방어력 감산%형 *(기본)* / 고정 감산형?
8. **땅속 유닛 규칙**: 단순(전용 대상만 피격) *(기본)* / 공격 시 부상 매복형?

→ "전부 기본값으로 진행" 한마디면 바로 다음 단계로 갑니다.
