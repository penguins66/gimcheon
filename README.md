# Gimcheon (가제)

1대1 라운드제 오토배틀러 (롤토체스 느낌). 준비 → 전투 → 정산을 반복하는 웹게임.
**현재 단계: M0 — 설계 문서 합의.** 코드는 합의 후 M1부터.

## 핵심 결정 (확정됨)
- 스택: **TypeScript + Vite**, 전투 Canvas / 준비 UI DOM, 게임 엔진 미사용.
- **시뮬레이션(순수·결정론) ↔ 렌더링 분리**, **데이터 주도** 콘텐츠 → 수정 쉽고 멀티 확장 가능.
- 1차: **솔로 모드(대 AI)** 만.
